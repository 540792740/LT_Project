####4. Sort
* arr.sort(); // returns [-1, -3, -5, 0, 1, 3]   <br> 
* **Ascend:** arr.sort((x, y) => x - y); // returns [-5, -3, -1, 0, 1, 3]
* **Descend:** arr.sort((x, y) => y - x); // returns [ 3, 1, 0, -1, -3, -5 ]

* parseInt(5 / 2) ->2       向下取整
* Math.round(5 / 2) ->3     四舍五入
